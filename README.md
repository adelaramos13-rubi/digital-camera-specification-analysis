# Digital Camera Specification Analysis (1994–2007)

## Project Overview

This project analyzes historical digital camera specifications between 1994 and 2007 using the 1000 Cameras Dataset [camera_dataset.csv](https://github.com/user-attachments/files/29865875/camera_dataset.csv)
[camera_dataset.csv](https://github.com/user-attachments/files/29865870/camera_dataset.csv)
from Kaggle. The objective was to explore how digital camera technology evolved over time and compare manufacturers based on camera resolution and weight.

The analysis was completed in Microsoft Excel using exploratory data analysis techniques, including PivotTables, scatter plots, line charts, and bar charts.

---

## Business Task

The primary objective of this project was to answer the following question:

> How did digital camera specifications evolve between 1994 and 2007, and how did manufacturers differ during this period?

To answer this question, the following business questions were explored:

- Which manufacturers produced cameras with the highest average effective megapixels?
- How did camera resolution change between 1994 and 2007?
- Do heavier cameras tend to have higher effective megapixel counts?
- How did average camera weight change between 1994 and 2007?
- Which manufacturers produced the heaviest cameras on average?

- Time Period: 1994–2007
- Records: 1,000 digital cameras
- Manufacturers: 21

---

## Key Findings

- Average camera resolution increased significantly between 1994 and 2007.
- Average camera weight decreased over the same period.
- Samsung had the highest average effective megapixel count.
- Sigma produced the heaviest cameras on average.
- Camera weight showed only a very weak relationship with effective megapixel count.
